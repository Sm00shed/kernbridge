# Kernbridge (KB)

A Rust-based configuration interface and gateway for OpenWrt.
Sits between you and your hardware. Replaces LuCI step by step,
without anyone noticing.

GPL v2. Community owned. Free.

---

## Principles

Kernbridge aspires to the principles of [Small Technology](https://small-tech.org/about#small-technology) by [Small Technology Foundation](https://small-tech.org), licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

- Easy to use
- Personal
- Private by default

- Peer to peer
- Interoperable
- Zero knowledge
- Non-commercial

- Inclusive
